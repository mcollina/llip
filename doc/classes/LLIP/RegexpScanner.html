Class LLIP::RegexpScanner
In: lib/llip/regexp_scanner.rb
Parent: RegexpAbstractScanner

It‘s a scanner for the parser RegexpParser. It has two kind of token: :char and :symbol.

  • char: every character.
  • symbol: . * + ( ) \ |

Constants

CHAR = LLIP::RegexpSpecification.new(:char)   It represents the regular expression ’.’
SYMBOL = LLIP::RegexpSpecification.new(:symbol)   It represents the regular expression ’(.|*|+|\(|\)|\|\|)’ so it matches the chars: . * + ( ) \ |
